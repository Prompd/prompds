# @prompd.io/excel-analytics

Excel-specialized analytics package inheriting from data-patterns with advanced Excel file processing, pivot table analysis, and formula evaluation.

## Features

- **Inherits from**: `@prompd.io/data-patterns@1.0.0`
- **Excel Processing**: Multi-worksheet analysis, structured tables, named ranges
- **Pivot Table Analysis**: Complete pivot table structure and insights analysis
- **Formula Evaluation**: Complex formula analysis and optimization suggestions
- **Chart Extraction**: Visualization analysis and effectiveness assessment
- **Business Intelligence**: Dashboard analysis and KPI identification
- **Data Validation**: Quality assessment and validation rule analysis

## Usage

```bash
prompd run excel-analytics.prmd \
  --data_source "financial_dashboard.xlsx" \
  --excel_processing_mode "comprehensive" \
  --pivot_table_analysis true \
  --formula_evaluation true
```

This package provides specialized Excel analysis capabilities built on the data-patterns foundation.
